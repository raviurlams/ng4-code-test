#Household App

1. git clone https://github.com/raviurlams/ng4-code-test.git
   
   open git bash on the project folder
  
2. Install the dependencies
    ```
    rm -rf node_modules dist tmp assets
    ng build  

    npm install
    ```

3. builds the app and run it on port 9000

    ng serve ---port 9000

    ```
